# List Users

Get a list of all your project's users. You can filter the results using queries such as search, sort, and pagination.

### API Information
- **Endpoint**: `/v1/users`
- **Method**: `GET`
- **Permissions**: `users.read`

### Usage Example
```json
{
  "name": "List Users",
  "endpoint": "/v1/users",
  "method": "GET"
}
```

Learn more about queries and listing resources at the [Appwrite Query documentation](https://appwrite.io/docs/queries).
