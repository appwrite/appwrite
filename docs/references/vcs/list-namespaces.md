Get a list of namespaces a VCS (Version Control System) installation can browse, such as the user's personal namespace and any groups they belong to. This is primarily useful for providers like GitLab, where a single OAuth authorization grants access to multiple namespaces with no per-installation selection step. For providers that already scope one installation to a single organization, this returns that organization as the only namespace. You can optionally filter namespaces using a search term.
